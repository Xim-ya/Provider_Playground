## 상태관리 라이브러리가 뭔가?

- 상태관리가 뭐고 Provider에서 제공하는 Provider 상태관리 기능이 무엇이 있는가?
- provider을 상태관리를 도와주는 라이브러리를 제공하지만 특정 상태관리 라이브러리 특정 방법을 강제하지는 않는다.
- 인스턴스를 쉽게 접근할 수 있고 UI를 업데이트 해주는 역할
